# amazon-review-collector
A slow web scraper designed to collect Amazon reviews buried in HTML.  For demonstration purposes only.
This scraper allows the colletion of a few reviews per page from a collection of Amazon links.  Most reviews are encrypted and are thus unavailable, but it should be able to collect and automatically translate up to 5-8 per page, with the appropriate star ratings and usernames usually preserved.  It runs at an extremely slow speed to be respectful and not waste Amazon's server resources, at around one link per minute.
